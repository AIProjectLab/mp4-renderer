## ✨ Key Features

### Core Rendering
- ✅ Hardware-accelerated H.264 (GPU encoding)
- ✅ Multiple resolutions (1080p, 1440p, 4K)
- ✅ Vertical/landscape and YouTube Shorts support
- ✅ Direct-to-disk streaming (unlimited video length)
- ✅ Flexible FPS & bitrate settings

### Subtitles & Audio
- ✅ Auto-subtitles with Groq Whisper API
- ✅ Auto-save SRT to /subtitles folder
- ✅ Audio speed control (0.5x-2.0x)
- ✅ 10 subtitle animation styles
